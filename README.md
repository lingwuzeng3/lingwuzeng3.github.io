# lingwuzeng3.github.io

一个简洁的个人网站，基于 GitHub Pages 托管。

## 访问地址

🌐 **https://lingwuzeng3.github.io**

## 站点结构

```
docs/
├── index.html          # 首页 - 个人介绍
├── tutorial.html       # GitHub Pages 部署教程
├── css/
│   ├── profile.css     # 主页样式（含暗色模式）
│   └── style.css       # 备用样式
├── js/
│   ├── profile.js      # 主页交互脚本
│   └── script.js       # 备用脚本
└── images/
    └── profile.jpg     # 头像图片
```

## 主要页面

| 页面 | 说明 |
|------|------|
| [index.html](docs/index.html) | 首页，展示个人介绍、技能、项目和联系方式 |
| [tutorial.html](docs/tutorial.html) | GitHub Pages 部署教程，包含快速开始和自定义域名配置 |

## 功能特性

- 响应式设计，适配各种屏幕尺寸
- 暗色模式自动适配（跟随系统主题）
- 平滑的页面滚动动画
- 移动端友好导航菜单

## 技术栈

- HTML5 + CSS3 + JavaScript
- Font Awesome 图标库
- 原生 CSS 暗色模式支持

## 部署说明

本项目使用 GitHub Pages 托管，`/docs` 文件夹作为部署源。

**部署来源设置**：Settings → Pages → Source → Deploy from a branch → `/docs`

详细教程请查看 [tutorial.html](docs/tutorial.html)
