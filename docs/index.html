GitHub Pages 部署指南

创建GitHub仓库

登录GitHub，创建一个新的仓库。如果创建用户站点，仓库名必须为username.github.io；如果是项目站点，可以任意命名。

上传网站文件

将你的静态网站文件（HTML、CSS、JS、图片等）上传到仓库。确保有一个index.html文件作为入口点。

启用GitHub Pages

进入仓库设置，找到"Pages"部分，选择部署来源（通常是main分支或/docs文件夹），然后保存。

访问你的网站

等待几分钟后，你的网站将在https://username.github.io（用户站点）或https://username.github.io/repository-name（项目站点）上线。
